## Requirement
  - Install mysql and redis

## Clone repositories
  - Clone repositories: git clone

## Installation
  - npm install

## Create env variables
  - JWT_SECRET
  - JWT_REFRESH
  - DATABASE_NAME
  - DATABASE_HOST
  - DATABASE_PORT
  - DATABASE_USERNAME
  - DATABASE_PASSWORD
  - REDIS_PORT
  - REDIS_HOST
  - EMAIL_SERVICE_HOST
  - EMAIL_SERVICE_PORT

## Run project
  - npm run start:dev
  - Go localhost:3000/swagger/ and run api
